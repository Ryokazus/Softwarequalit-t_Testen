2.) a) Vergleich ArrayQueue mit ArrayStack, Erläuterung n, start und end: +

bei Queue hingegen auf das erste. Bei pop wird bei beiden das letzte Element genommen. Der
Während der Stack nicht geprüft wird ob er voll ist, wird bei Queue dies geprüft.

n -> Länge des Arrays + 
start -> erste eingefügtes Element, wo Elemente weggenommen werden können +
end -> letztes eingefügtes Element, wo neue hinzugefügt werden
